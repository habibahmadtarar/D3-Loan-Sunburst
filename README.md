# Loan Approval Sunburst Chart

This project visualizes loan approval data for the USA and Canada using an interactive **sunburst chart** built with **D3.js**.

## Features
- Interactive sunburst chart with clickable slices for zoom-in effect.
- Tooltip displaying category names and values on hover.
- Responsive color scheme to differentiate parent and child nodes.
- Smooth animated transitions on slice click.

## Technologies Used
- **HTML5**: Structure of the page.
- **CSS3**: Styling of chart and tooltip.
- **JavaScript (ES6)**: Data handling, DOM manipulation, and D3.js integration.
- **D3.js v7**: Data-driven visualizations and SVG path generation.
- **JSON**: Structured data source (`data/data.json`).
